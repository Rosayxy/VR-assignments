# VR assignments
THU CST 虚拟现实技术课程的3次小作业。  
第一次作业是实现顶点坐标变换过程中模型变换、视角变换、投影变换的变换矩阵计算函数     
第二次作业实现空间立体声    
第三次作业用 Unity 编程，实现 Meta Quest Link 上的光线追踪技术（最终效果挺帅的，让人想起火影里面的式纸之舞hhh），因为该文件包太大了，所以用清华云盘上传，链接如下：
